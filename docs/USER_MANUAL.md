## 1. 系统概述

入党文档管理系统是一个用于管理党员发展全流程的Web应用系统。系统支持：
- 申请人提交入党申请及相关文档
- 书记审核申请人提交的材料
- 管理员管理用户、支部和文档模板
- 完整的入党发展5阶段26步骤流程跟踪

## 2. 系统启动

### 2.1 环境要求
- Python 3.8+
- SQLite3
- 现代浏览器（Chrome、Firefox、Edge）

### 2.2 启动步骤

**步骤1：进入项目目录**
```bash
cd d:\Users\Administrator\CPCWebIII
```

**步骤2：激活虚拟环境（如适用）**
```bash
# Windows
.\venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

**步骤3：启动服务器**
```bash
python run.py
```

**步骤4：访问系统**
- 打开浏览器访问：http://127.0.0.1:5003
- 或局域网访问：http://服务器IP:5003

### 2.3 首次启动

首次启动时，系统会自动：
1. 创建数据库文件 `backend/data/cpc.db`
2. 初始化数据库表结构
3. 创建默认管理员账户

## 3. 用户角色与权限

系统包含三种用户角色：

| 角色 | 中文名 | 权限说明 |
|------|--------|----------|
| admin | 管理员 | 系统最高权限，可管理用户、支部、模板，处理最终审批 |
| secretary | 书记 | 支部书记权限，可审核本支部申请人的文档和申请 |
| applicant | 申请人 | 普通申请人权限，可提交申请、上传文档、查看进度 |

### 权限矩阵

| 功能 | 管理员 | 书记 | 申请人 |
|------|:------:|:----:|:------:|
| 用户管理 | ✅ | ❌ | ❌ |
| 支部管理 | ✅ | ❌ | ❌ |
| 模板管理 | ✅ | ❌ | ❌ |
| 审批管理 | ✅ | ❌ | ❌ |
| 查看申请人 | ✅ | ✅(本支部) | ❌ |
| 审核文档 | ✅ | ✅(本支部) | ❌ |
| 提交申请 | ❌ | ❌ | ✅ |
| 上传文档 | ❌ | ❌ | ✅ |
| 查看进度 | ❌ | ❌ | ✅ |
| 导出文档 | ✅ | ✅ | ✅(自己的) |

## 5. 申请人操作指南

### 5.1 申请人首页

登录后，申请人将看到：
- **当前阶段**：显示入党申请所处的阶段
- **进度时间线**：展示各步骤的完成情况
- **待办事项**：需要完成的任务列表
- **通知消息**：系统消息和审批结果通知

### 5.2 查看发展进度

1. 点击导航栏【发展进度】
2. 查看完整的入党发展时间线
3. 每个阶段显示：
   - 阶段名称和状态
   - 完成时间
   - 相关文档

### 5.3 上传文档

1. 点击导航栏【我的文档】
2. 点击"上传文档"按钮
3. 选择文档类型：
   - 入党申请书
   - 思想汇报
   - 自传
   - 其他材料
4. 选择文件（支持 .doc, .docx, .pdf）
5. 填写文档说明
6. 点击"上传"

### 5.4 查看审批结果

- 文档提交后，可在【我的文档】页面查看审批状态
- 状态说明：
  - **待审核**：书记尚未审核
  - **已通过**：审核通过
  - **已驳回**：审核未通过，需重新提交

### 5.5 被驳回后重新提交

1. 在【我的文档】找到被驳回的文档
2. 查看驳回原因
3. 修改文档后重新上传
4. 等待再次审核

## 6. 书记操作指南

### 6.1 书记工作台

登录后，书记将看到：
- **待审核数量**：等待审核的申请人/文档数
- **本支部统计**：支部内申请人概况
- **待办列表**：需要处理的审批任务

### 6.2 管理申请人

1. 点击导航栏【申请人管理】
2. 查看本支部所有申请人列表
3. 可按状态、阶段筛选
4. 点击申请人查看详情

### 6.3 审核文档

1. 点击导航栏【文档审核】
2. 查看待审核文档列表
3. 点击文档查看详情
4. 选择操作：
   - **通过**：文档符合要求
   - **驳回**：填写驳回原因，要求重新提交

### 6.4 审批流程步骤

1. 查看申请人提交的材料
2. 核实材料完整性和准确性
3. 填写审批意见
4. 提交审批决定
5. 系统自动通知申请人

## 7. 管理员操作指南

### 7.1 管理首页

登录后，管理员将看到：
- **系统统计**：用户数、支部数、申请数等
- **最近申请**：最新的入党申请列表
- **快捷入口**：常用功能快速访问

### 7.2 用户管理

**创建用户：**
1. 点击【用户管理】→【新增用户】
2. 填写信息：
   - 用户名（必填，唯一）
   - 姓名（必填）
   - 工号（可选）
   - 角色（管理员/书记/申请人）
   - 所属支部
3. 点击"保存"

**编辑用户：**
1. 在用户列表找到目标用户
2. 点击"编辑"按钮
3. 修改信息
4. 点击"保存"

**禁用/启用用户：**
- 点击"禁用"按钮可暂时禁止用户登录
- 点击"启用"按钮恢复用户访问权限

### 7.3 支部管理

**创建支部：**
1. 点击【支部管理】→【新增支部】
2. 填写支部名称和描述
3. 点击"保存"

**分配书记：**
1. 在【用户管理】创建书记账户
2. 设置角色为"书记"
3. 选择所属支部

### 7.4 模板管理

**上传文档模板：**
1. 点击【模板管理】→【上传模板】
2. 填写模板信息：
   - 模板名称
   - 适用阶段（1-5阶段）
   - 步骤代码（如L1, L2等）
   - 选择文件
3. 点击"上传"

**更新模板：**
1. 找到需要更新的模板
2. 点击"编辑"
3. 上传新版本文件
4. 保存后，新申请将使用新模板

### 7.5 审批管理

**处理最终审批：**
1. 点击【审批管理】
2. 查看待审批申请
3. 核实申请人已完成所有必要步骤
4. 做出审批决定

## 8. 入党申请流程

### 8.1 五阶段二十六步骤

```
阶段1: 申请人 → 入党积极分子
├── 步骤1: 申请人
├── 步骤2: 入党积极分子
└── ...

阶段2: 入党积极分子 → 发展对象
├── 步骤3: 确定发展对象
├── 步骤4: 政治审查
└── ...

阶段3: 发展对象 → 预备党员
├── 步骤5: 确定预备党员
└── ...

阶段4: 预备党员
├── 步骤6: 预备期培养
└── ...

阶段5: 预备党员 → 正式党员
├── 步骤7: 转正
└── ...
```

### 8.2 完整审批流程图

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   申请人    │───▶│    书记    │───▶│   管理员   │
│ 上传文档    │    │  审核文档   │    │  最终审批   │
└─────────────┘    └─────────────┘    └─────────────┘
       │                  │                   │
       │    ┌─────────────┴─────────────┐     │
       │    │                           │     │
       ▼    ▼                           ▼     ▼
   通过→下一阶段              驳回→重新提交    完成归档
```

### 8.3 文档要求

| 阶段 | 必需文档 |
|------|----------|
| 阶段1 | 入党申请书、个人简历、自传 |
| 阶段2 | 思想汇报(每季度)、培训结业证书 |
| 阶段3 | 政审材料、函调材料、群众座谈会记录 |
| 阶段4 | 预备党员考察表、季度思想汇报 |
| 阶段5 | 转正申请书、预备期总结 |

## 9. 常见问题

### Q1: 忘记密码怎么办？
**A:** 联系系统管理员重置密码，重置后默认密码为 `123456`。

### Q2: 上传文档失败？
**A:** 检查以下几点：
- 文件格式是否为 .doc, .docx 或 .pdf
- 文件大小是否超过限制（默认10MB）
- 网络连接是否正常

### Q3: 审批被驳回怎么办？
**A:** 
1. 查看驳回原因
2. 按要求修改文档
3. 重新上传
4. 等待再次审核

### Q4: 如何查看申请进度？
**A:** 申请人登录后点击【发展进度】，可查看完整的审批时间线。

### Q5: 书记看不到申请人？
**A:** 确保：
1. 书记账户已分配支部
2. 申请人属于同一支部
3. 书记角色设置正确

### Q6: 如何导出文档？
**A:** 
1. 管理员/书记：在申请人详情页点击"导出文档"
2. 申请人：在【我的文档】点击"导出"
